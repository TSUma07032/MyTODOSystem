# My Ultimate TODO
> Notionとの併用を前提にタスクの入出力を簡単にしたTODOリスト

## 問題意識
既存のTODOリストは、モチベーション維持とタスク管理を問題意識として掲げています。しかし、タスクの追加や完了が大変という問題を抱えています。
そこで、モチベーション維持のための機能を切り捨てたり、私の作業環境に即したTODOリストを作れば、タスクの管理を圧倒的に楽にできるのではないかと考えました。

今回はNotionのチェックリストに着目しました。Notionは、チェックリストを以下のようなマークダウンを通じて扱っています。

```
- [ ]  TODO改修
  - [ ]  デイリー機能、ウィークリー機能
    - [ ] デイリーの難易度調整機能
  - [×] jsonに移行
```

つまり、正規化処理等を活用すれば、Notionのチェックリスト経由で容易なタスク管理を実現できるのではないかと考えたのです！

## ビルドの仕方メモ
ビルドする

npm run build
npm run preview

デベロッパーツールを開く

アプリの画面で F12 キー（または Ctrl + Shift + I）を押して。

「Application」タブを選択

上のメニューにある 「Application（アプリケーション）」 タブをクリック。

「Service Workers」を選択

左側のサイドバーにある 「Service Workers」 をクリック。

「Unregister（登録解除）」を実行

画面中央に表示されている sw.js などの項目の右端にある 「Unregister」 という青い文字をクリック。

ページをリロード（再読み込み）

Ctrl + F5（または Ctrl + R）でページを更新して。
